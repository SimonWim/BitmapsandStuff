Embedded Bitmaps

These images are stored directly in the HTML file using Base64 data URIs.

Theres also a selection of bitmaps which you can access as binary files via their respectives links. You may want to invert the files to save some power on your display

The links to these bitmaps are just: "https://simonwim.github.io/BitmapsandStuff/xxxbitmap.bin"

The available bitmaps and their names are found in the repository of this

site

You can dynamically access these images via their `id` attributes from your ESP's code or JavaScript.

For the bitmaps you can also just directly go to the links as described above and your microcontroller should be able to process the binary files
